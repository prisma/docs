## In this section

Scaffold a new Prisma project with a new database using the `prisma init --prompt` command:

```npm
npx prisma init --prompt "Simple habit tracker application"
```

This will generate a Prisma schema based on the prompt and deploy it to a fresh [Prisma Postgres](/docs/v6/postgres) instance.

## Prisma MCP server

Prisma provides its own [Model Context Protocol](https://modelcontextprotocol.io/introduction) (MCP) server that lets you manage Prisma Postgres databases, model database schemas and chat through migrations. Learn more about it [here](/docs/v6/postgres/integrations/mcp-server).
