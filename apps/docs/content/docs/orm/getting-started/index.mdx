# Getting Started with Prisma ORM

Prisma ORM is a next-generation Node.js and TypeScript ORM that provides type-safe database access, migrations, and a powerful query API.

## Choose Your Path

### Start Fresh

New to Prisma? Start with our quickstart guides:

- [**Quickstart with Prisma Postgres**](/docs/v7/getting-started/prisma-orm/quickstart/prisma-postgres) - Fastest way to get started with Prisma's managed PostgreSQL
- [**Quickstart with PostgreSQL**](/docs/v7/getting-started/prisma-orm/quickstart/postgresql) - Use your own PostgreSQL database
- [**Quickstart with SQLite**](/docs/v7/getting-started/prisma-orm/quickstart/sqlite) - Perfect for local development
- [**Quickstart with MySQL**](/docs/v7/getting-started/prisma-orm/quickstart/mysql) - Use MySQL database
- [**Quickstart with MongoDB**](/docs/v7/getting-started/prisma-orm/quickstart/mongodb) - Use MongoDB database

### Add to Existing Project

Already have a project? Add Prisma ORM to your existing application:

- [**Add Prisma to existing project**](/docs/v7/getting-started/prisma-orm/add-to-existing-project/prisma-postgres) - Integrate Prisma ORM into your current codebase

## What's Next?

After getting started, explore these key topics:

- [**Prisma Schema**](/docs/v7/orm/prisma-schema) - Learn how to define your data model
- [**Prisma Client**](/docs/v7/orm/prisma-client) - Master the query API
- [**Prisma Migrate**](/docs/v7/orm/prisma-migrate) - Handle database migrations
