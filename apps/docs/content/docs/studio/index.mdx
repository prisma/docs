---
title: "Prisma Studio"
description: "A visual database editor for viewing and managing your data with Prisma"
---

Prisma Studio is a powerful visual database editor that lets you view and manipulate data directly in your browser. The latest version is SQL-driven and works with or without Prisma ORM, making it a versatile tool for database management.

## Key Features

- **Database-Agnostic**: Works with PostgreSQL, MySQL, and SQLite
- **No Prisma Required**: Can be used with any database using a connection string
- **Real-time Schema Introspection**: Automatically detects your database structure
- **Intuitive UI**: Edit data in a spreadsheet-like interface
- **Embeddable**: Integrate Studio directly into your Next.js applications

## Getting Started

```bash
# With Prisma project
npx prisma studio

# With direct database connection
npx prisma studio --url="postgresql://user:password@localhost:5432/dbname"
```
