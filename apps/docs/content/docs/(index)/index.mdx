[**Prisma ORM**](/docs/orm) is an open-source ORM that provides fast, type-safe access to Postgres, MySQL, SQLite, and other databases, and runs smoothly across Node.js, Bun, and Deno.

```npm
npx prisma init --db
```


[**Prisma Postgres**](/docs/postgres) is a fully managed PostgreSQL database that scales to zero, integrates with [Prisma ORM](/docs/orm) and [Prisma Studio](/docs/studio), and includes a [generous free tier](https://www.prisma.io/pricing).

```npm
npx create-db
```
