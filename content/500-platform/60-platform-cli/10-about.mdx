This documentation provides details on how to access the Prisma Data Platform using the Prisma CLI. Follow the [installation guide](/orm/tools/prisma-cli#installation) for the Prisma CLI to start using it.

The integration, currently in [Early Access](/platform/maturity-levels#early-access), extends the functionality of the Prisma CLI, enabling you to manage your Platform projects. For the complete list of available commands, navigate to the [Commands page](/platform/platform-cli/commands).
