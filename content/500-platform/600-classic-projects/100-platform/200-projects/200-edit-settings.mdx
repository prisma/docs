<TopBlock>

In the project settings, you can change the project display name and the project handle.

</TopBlock>

## Prerequisites and considerations

You are the [Project owner](/platform/classic-projects/platform/members/roles-permissions) or have the [Admin](/platform/classic-projects/platform/members/roles-permissions) role.

## Steps

1. In the upper left, click your project handle and select **Project settings**.
   <img
     src="../../images/common/open-project-settings.png"
     alt="Prisma Data Platform - Edit project settings - Change the project name and the project URL handle"
     width="350px"
   />
2. In **Project Name**, change the project name.

   <Admonition>

   **Note**<br /><br />

   The project name appears in the following places in the Prisma Data Platform:<br /><br />
   &nbsp;• [_Projects_](https://cloud.prisma.io/projects) page<br />
   &nbsp;• [_Billing_](https://cloud.prisma.io/billing) page<br />
   &nbsp;• **Projects** drop-down in the upper left

   </Admonition>

3. In **Project Handle**, change the project handle.

   <Admonition>

   **Note**<br /><br />

   The project handle defines the project part of the URL in the Prisma Data Platform.<br /><br />

   `https://cloud.prisma.io/projects/<project-handle>`<br /><br />

   For example, if your project handle is `nextjs-db`, your project URL is `https://cloud.prisma.io/projects/nextjs-db`

   </Admonition>

## Result

The changes to the project name and handle are saved.
