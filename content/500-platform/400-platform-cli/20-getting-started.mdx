<TopBlock>

To get started, ensure you have the Prisma Client updated to version `5.7.0` or later. This is necessary to access the Platform through the Prisma CLI.

</TopBlock>

<Admonition type="warning">

When using commands, always start with prisma platform and include the --early-access flag to enable the use of the Prisma Data Platform whilst still in early access.

</Admonition>
