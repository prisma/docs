## Prisma config reference

The Prisma config file is a TypeScript file that allows you to configure the Prisma client without using a `.env` file.

The config file is located in the `prisma` directory and is named `prisma.config.ts`.

## Prisma config example

```ts
import { defineConfig } from '@prisma/config/src'

export default defineConfig({
  experimental: true,
})
```
