##  Prerequisites

In order to successfully complete this guide, you need:

* Node.js installed on your machine
* A MSSQL database running
* The `sqlcmd` command line tool (alternatively, Microsoft SQL Server Management studio)

Make sure you have your database connection URL (that includes your authentication credentials) at hand!
