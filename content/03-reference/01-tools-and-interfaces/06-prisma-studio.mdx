## Overview

Prisma Studio is a visual editor for your database. To run Prisma Studio, run the following command in your terminal:

```console
npx prisma studio --experimental
```

> **Note**: Prisma Studio replaces Prisma Admin, and is only compatible with Prisma 2. Be aware that Prisma Studio is currently in an **experimental state**.
