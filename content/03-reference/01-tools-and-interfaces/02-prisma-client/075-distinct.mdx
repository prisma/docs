## Examples 

The following examples are based on a sample schema:

```prisma
  provider             = "prisma-client-js"
  experimentalFeatures = ["distinct", "middlewares"]

datasource db {
  provider = "mysql"

  authorId  Int?
  content   String?
  id        Int     @default(autoincrement()) @id
  published Boolean @default(false)
  title     String
  user      User?   @relation(fields: [authorId], references: [id])

  @@index([authorId], name: "authorId")
}

model User {
  email           String  @unique
  id              Int     @default(autoincrement()) @id
  name            String?
  posts           Post[]
  extendedProfile Json?
  role            Role    @default(USER)
  city            String?
  country         String?
}

enum Role {
  ADMIN
  USER
  MODERATOR
}
```

### Select distinct on a single field

The following example returns all distinct `city` fields:

```ts
const distinctCities = await prisma.user.findMany({
  select: {
    city: true,
    country: true,
  },
  distinct: ["city"],
});
```

```js no-lines
[
  { city: 'Paris', country: 'France' },
  { city: 'Lyon', country: 'France' }
]
```
### Select distinct on multiple fields

Select distinct city AND country

```ts 
const distinctCitiesAndCountries = await prisma.user.findMany({
  select: {
    city: true,
    country: true,
  },
  distinct: ["city", "country"],
});
```

```js no-lines
[
  { city: 'Paris', country: 'France' },
  { city: 'Paris', country: 'Denmark' },
  { city: 'Lyon', country: 'France' }
]
```
