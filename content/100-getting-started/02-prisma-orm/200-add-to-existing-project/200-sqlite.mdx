# Add Prisma ORM to existing project with SQLite

Learn how to integrate Prisma ORM with SQLite into your existing Node.js or TypeScript project. SQLite is perfect for development and small applications.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client @prisma/adapter-better-sqlite3
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure for SQLite

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider   = "prisma-client"
  output     = "../generated/prisma"
  engineType = "client"
}

datasource db {
  provider = "sqlite"
  url      = "file:./dev.db"
}
```

## 4. Create your schema and migrate

```terminal
npx prisma migrate dev --name init
```
