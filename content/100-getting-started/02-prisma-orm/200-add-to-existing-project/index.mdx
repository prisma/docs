# Add Prisma ORM to existing project

Learn how to integrate Prisma ORM into your existing Node.js or TypeScript project. Choose your database below to get started with a step-by-step guide tailored to your setup.

## Database-specific guides

### Recommended

**[Prisma Postgres](/getting-started/prisma-orm/add-to-existing-project/prisma-postgres)** - Managed database with built-in connection pooling and caching

### Other databases

- **[SQLite](/getting-started/prisma-orm/add-to-existing-project/sqlite)** - Perfect for development and small applications
- **[PostgreSQL](/getting-started/prisma-orm/add-to-existing-project/postgresql)** - Popular open-source relational database
- **[MySQL](/getting-started/prisma-orm/add-to-existing-project/mysql)** - Widely-used relational database
- **[SQL Server](/getting-started/prisma-orm/add-to-existing-project/sql-server)** - Microsoft's enterprise database
- **[PlanetScale](/getting-started/prisma-orm/add-to-existing-project/planetscale)** - Serverless MySQL platform
- **[CockroachDB](/getting-started/prisma-orm/add-to-existing-project/cockroachdb)** - Distributed SQL database
- **[MongoDB](/getting-started/prisma-orm/add-to-existing-project/mongodb)** - Document-based NoSQL database

## What you'll learn

In these guides, you'll learn how to:

- Install Prisma ORM in your existing project
- Configure your database connection
- Set up your Prisma schema
- Generate a type-safe Prisma Client
- Integrate Prisma Client into your existing codebase
- Handle database introspection and migrations

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed on your machine
- Basic knowledge of JavaScript or TypeScript
- Access to a database (or we'll help you set one up)

## Integration patterns

Each guide includes examples for popular frameworks and patterns:

- **Express.js** - REST API integration
- **Next.js** - API routes and server components
- **Standalone scripts** - Database utilities and migrations
- **Testing** - Setting up Prisma Client for tests

## Next steps

After integrating Prisma ORM:

- Learn about [database migrations](/orm/prisma-migrate)
- Set up [Prisma Studio](/orm/tools/prisma-studio) for visual database management
