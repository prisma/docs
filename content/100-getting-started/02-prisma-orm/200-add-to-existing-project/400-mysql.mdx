# Add Prisma ORM to existing project with MySQL

Learn how to integrate Prisma ORM with MySQL into your existing Node.js or TypeScript project.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- A MySQL database accessible
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client @prisma/adapter-mariadb
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure your database connection

Update `.env` with your MySQL connection:

```env file=.env
DATABASE_URL="mysql://username:password@localhost:3306/mydb"
```

## 4. Configure for MySQL

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider   = "prisma-client"
  output     = "../generated/prisma"
  engineType = "client"
}

datasource db {
  provider = "mysql"
  url      = env("DATABASE_URL")
}
```

## 5. Introspect existing database (optional)

If you have existing data:

```terminal
npx prisma db pull
```

Or create new schema and migrate:

```terminal
npx prisma migrate dev --name init
```
