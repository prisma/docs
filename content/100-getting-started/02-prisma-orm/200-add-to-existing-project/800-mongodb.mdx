# Add Prisma ORM to existing project with MongoDB

Learn how to integrate Prisma ORM with MongoDB into your existing Node.js or TypeScript project.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- A MongoDB database accessible
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure your database connection

Update `.env` with your MongoDB connection:

```env file=.env
DATABASE_URL="mongodb+srv://username:password@cluster.mongodb.net/mydb"
```

## 4. Configure for MongoDB

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider = "prisma-client"
  output   = "../generated/prisma"
}

datasource db {
  provider = "mongodb"
  url      = env("DATABASE_URL")
}
```

## 5. Introspect existing database (optional)

If you have existing collections:

```terminal
npx prisma db pull
```

Or push new schema:

```terminal
npx prisma db push
```
