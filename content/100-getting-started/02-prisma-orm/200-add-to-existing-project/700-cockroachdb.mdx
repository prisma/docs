# Add Prisma ORM to existing project with CockroachDB

Learn how to integrate Prisma ORM with CockroachDB into your existing Node.js or TypeScript project.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- A [CockroachDB](https://www.cockroachlabs.com/) database accessible
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client @prisma/adapter-pg
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure your database connection

Update `.env` with your CockroachDB connection:

```env file=.env
DATABASE_URL="postgresql://username:password@host:26257/mydb?sslmode=require"
```

## 4. Configure for CockroachDB

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider   = "prisma-client"
  output     = "../generated/prisma"
  engineType = "client"
}

datasource db {
  provider = "cockroachdb"
  url      = env("DATABASE_URL")
}
```

## 5. Introspect existing database (optional)

If you have existing data:

```terminal
npx prisma db pull
```

Or create new schema and migrate:

```terminal
npx prisma migrate dev --name init
```
