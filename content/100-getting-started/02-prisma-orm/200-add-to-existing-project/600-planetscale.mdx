# Add Prisma ORM to existing project with PlanetScale

Learn how to integrate Prisma ORM with PlanetScale into your existing Node.js or TypeScript project.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- A [PlanetScale](https://planetscale.com) database
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client @prisma/adapter-planetscale
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure your database connection

Update `.env` with your PlanetScale connection:

```env file=.env
DATABASE_URL="mysql://username:password@host.connect.psdb.cloud/mydb?sslaccept=strict"
```

## 4. Configure for PlanetScale

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider   = "prisma-client"
  output     = "../generated/prisma"
  engineType = "client"
}

datasource db {
  provider     = "mysql"
  url          = env("DATABASE_URL")
  relationMode = "prisma"
}
```

## 5. Push schema to PlanetScale

PlanetScale uses schema branching instead of migrations:

```terminal
npx prisma db push
```
