# Add Prisma ORM to existing project with SQL Server

Learn how to integrate Prisma ORM with SQL Server into your existing Node.js or TypeScript project.

## Prerequisites

- An existing Node.js or TypeScript project
- [Node.js](https://nodejs.org/en/) installed
- A SQL Server database accessible
- Basic knowledge of JavaScript or TypeScript

## 1. Install Prisma ORM

```terminal
npm install prisma --save-dev
npm install @prisma/client @prisma/adapter-mssql
```

## 2. Initialize Prisma ORM

```terminal
npx prisma init --output ../generated/prisma
```

## 3. Configure your database connection

Update `.env` with your SQL Server connection:

```env file=.env
DATABASE_URL="sqlserver://localhost:1433;database=mydb;user=username;password=password;encrypt=true"
```

## 4. Configure for SQL Server

Update `prisma/schema.prisma`:

```prisma file=prisma/schema.prisma
generator client {
  provider   = "prisma-client"
  output     = "../generated/prisma"
  engineType = "client"
}

datasource db {
  provider = "sqlserver"
  url      = env("DATABASE_URL")
}
```

## 5. Introspect existing database (optional)

If you have existing data:

```terminal
npx prisma db pull
```

Or create new schema and migrate:

```terminal
npx prisma migrate dev --name init
```
